# JSC

这是一个开发FP(For Android)前的JavaScript引擎技术验证Demo，采用C++实现。示例中calculation函数实现了一个简单的通过JSC环境计算的四则运算。而callObject则是在JSC中构建自定义全局对象、对象属性和对象函数从而实现JNI的调用(即Android调用C++).
